# Third Party

The following services are recommended and compatible with Unownhash:

## ReactMap

<Image
  src="/images/reactmap.png"
  alt="ReactMap logo"
  width={250}
  height={250}
/>

ReactMap is a map written in React allowing users to find Pokemon based on information in your database.

<SourceLink
  git_href="https://github.com/WatWowMap/ReactMap"
  docs_href="https://github.com/WatWowMap/ReactMap/wiki"
/>

## PoracleJS

<Image
  src="/images/poraclejs.svg"
  alt="PoracleJS logo"
  width={250}
  height={250}
/>

A bot that allows for the configuration of Discord or Telegram alerts.

<SourceLink
  git_href="https://github.com/KartulUdus/PoracleJS"
  docs_href="https://muckelba.github.io/poracleWiki"
/>

## Koji

<Image src="/images/koji.png" alt="Koji logo" width={250} height={250} />

Koji is an automation tool that can streamline mapping new and existing areas as efficiently as possible. It includes a powerful route manager written in React & Rust.


<SourceLink
  git_href="https://github.com/TurtIeSocks/Koji"
  docs_href="https://koji.vercel.app"
/>
