# Contributing

We welcome all contributions. We are an Open Source project, and as such, we know time is one of the most valuable resources an individual can give is time. Therefore, the team greatly appreciates documentation fixes, helping other community members, reporting bugs, testing features, and pull requests!

## Code

We use the standard GitHub flow of merging changes. First, please open a pull request on the correct Git repo. Then, when someone has time, we will review the change, offer comments, and either merge or reject the change.
