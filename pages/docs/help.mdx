# Getting Help

If you are facing a problem, we encourage you to reach out in one of our Discord `#help` channels; the invite link is below. This workflow allows the larger community to answer questions instead of throwing the burden of support on our maintainers.

## Found a bug

    <Callout type="info" emoji="ℹ️">
    Before reporting a bug, please ensure you are running the latest version.
    </Callout>

If your issue is still present, we encourage you to [open an issue](https://github.com/unownhash) on the correct GitHub project! Please thoroughly describe, attach logs, and provide steps to replicate the issue.
