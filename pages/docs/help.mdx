# Getting Help

If you are running into a problem, we encourage you to reach out in one of the `#help` channels in discord. This allows the larger community to answer questions instead of throwing the burden of support on our maintainers.

## Found a bug

    <Callout type="info" emoji="ℹ️">
    Please update to the latest version if you aren't already. This is the best way to make sure you have all the latest fixes.
    </Callout>

If your issue is still present, we encourage you to [open an issue](https://github.com/unownhash) on the correct GitHub project! Please be through, provide logs, and try to provide steps on how we can replicate the issue.
